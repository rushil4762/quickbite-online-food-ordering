# QuickBite

QuickBite is a modern Food Ordering Web Application developed using Django.  
The project provides a smooth and user-friendly platform for customers to browse food items, manage carts, and place orders online.

---

## Features

- User Authentication (Login/Register)
- Food Browsing and Categories
- Add to Cart Functionality
- Order Placement System
- Admin Dashboard
- Order Management
- Responsive User Interface
- Database Integration with SQLite

---

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap

### Backend
- Python
- Django

### Database
- SQLite
